## Getting Started

### First Time Setup
1. **Onboarding**: Follow the 3-step guide
2. **Add First Transaction**: Tap the + button
3. **Set Budget** (optional): Go to Settings > Budget Goals
4. **Customize Categories**: Settings > Manage Categories

### Daily Use
1. **Add transactions** as they happen (takes 10 seconds!)
2. **Check dashboard** to see spending patterns
3. **Review reports** weekly/monthly
4. **Adjust budgets** as needed

## Troubleshooting

### App Not Loading?
- Pull down to refresh
- Check internet (if using cloud sync)
- Restart the app

### Can't Find Transaction?
- Use the search feature
- Check filters (All/Income/Expense)
- Sort by different criteria

### Data Backup
- Export data regularly via Settings
- Keep backups in cloud storage
- Use both CSV (for Excel) and JSON (for backup)
